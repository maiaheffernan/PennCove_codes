# PennCove_codes
This repository contains the codes used to program sensors, retrieve data, and perform analysis on data collected from Penn Cove sampling. Each sensor has its own directory in this repository. Code is written primarily in Matlab and Python. 

### Directory structure

- **Sensors:** read-in and preliminary processing scripts
- **Processing:** higher-level data processing and cleaning
- **Analysis:** scripts dedicated to specific analyses and hypothesis testing
- **Figures:** primary/main figures from analyses
